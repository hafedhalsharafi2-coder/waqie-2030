مشروع واقع الرؤية

فكرة المشروع:
صحيفة إلكترونية مهنية تتابع مشاريع رؤية السعودية 2030 وتشرح أثرها على المجتمع والاقتصاد والتقنية وجودة الحياة بأسلوب صحفي واضح.

فريق العمل - المجموعة العاشرة:
- حلا المطيري
- العنود السحيمي
- سميه خلاف
- ريم مشعان الحربي
- رواف الرحيلي

طريقة الرفع على GitHub Pages:
1) أنشئ Repository جديد واجعله Public.
2) ارفع الملفات نفسها، وليس ملف ZIP فقط.
3) تأكد أن ملف الصفحة الرئيسية اسمه index.html.
4) بعد الرفع: Settings ثم Pages.
5) اختر Branch: main ثم Save.
6) انتظر قليلًا وسيظهر رابط الموقع.

الملفات:
- index.html: الصفحة الرئيسية
- projects.html: مشاريع الرؤية
- economy.html: الاقتصاد
- technology.html: التقنية
- quality.html: جودة الحياة
- contact.html: تواصل معنا
- style.css: التنسيق العام
- script.js: تفاعل نموذج التواصل
